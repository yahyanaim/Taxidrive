# Wingy Site

A static-only Next.js 14 application.

## Installation

```bash
pnpm install
```

## Development

To run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To build the application for production:

```bash
pnpm build
```

## Notes

This application is static-only and has no backend integration.
